Enable WooCommerce customers to effortlessly manage their personal wishlists, improving their shopping experience!

== Description ==

OLLITS Wishlist for WooCommerce empowers your customers with seamless wishlist functionality, enabling them to effortlessly curate and manage their favorite products. With just a click, users can add or remove items from their personal wishlists, creating a tailored shopping experience that keeps them coming back.

== Key Features ==

* Designate a specific page for displaying the wishlist.
* Select the location for the ‘Add to Wishlist’ shortcode.
* Show the ‘Remove from Wishlist’ button for products already added to the wishlist.
* Enable the ‘Add to Wishlist’ button on the Shop page for easy access.
* Customize which columns appear in the wishlist table to suit your preferences.
* Support for product variations — when users select specific attributes like color or size, those details are saved in the wishlist.

== Important Note for Site Administrators ==

To ensure a smooth experience, we recommend creating a complete backup of your site before installing and activating this plugin. This will help protect your data and allow for easy recovery if needed.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ollits-woo-wishlist` directory, or install the plugin through the WordPress plugins screen directly.
2. Ensure that WooCommerce is installed and activated.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the **Settings->Advanced Wishlist** screen to configure the plugin.

== Frequently Asked Questions ==

= How do I use the plugin? =

To use the wishlist feature, simply add the shortcode `[ollits_wishlist]` to any page or post where you want the wishlist to appear. You can also configure additional settings through the plugin's settings page.
